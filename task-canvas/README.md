# Trello Clone (Frontend)

A React and Vite frontend for a Trello-style Kanban board application. This project provides a lightweight task management experience with drag-and-drop boards, cards, and workspace navigation inspired by Trello.

## Features

- Kanban board UI with columns and cards
- Create, open, and manage boards
- Drag-and-drop card organization
- Responsive design using Tailwind CSS
- Modular UI components for cards, lists, and navigation

## Tech Stack

- React with TypeScript
- Vite build tool
- Tailwind CSS for styling
- `@hello-pangea/dnd` for drag-and-drop interaction
- Lucide icons for UI elements

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal to view the app.

## Notes

This repository contains only the frontend application in the `task-canvas` folder. It is designed to be paired with a backend API if needed, but it can also be used standalone for UI development and prototyping.
